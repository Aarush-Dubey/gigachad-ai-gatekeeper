# Deployment Guide

This app is split into two parts:
1. **Frontend (`client/`)**: Static HTML/JS. Host on Vercel/Netlify.
2. **Backend (`server/`)**: FastAPI. Host on Render/Railway.

## 1. Database (Supabase)
1. Create a project at [supabase.com](https://supabase.com).
2. Run this SQL in the SQL Editor:
   ```sql
   create table candidates (
     id bigint generated by default as identity primary key,
     created_at timestamp with time zone default timezone('utc'::text, now()) not null,
     name text,
     email text,
     student_id text,
     timestamp text,
     synced boolean default false
   );
   ```
3. Get your `SUPABASE_URL` and `SUPABASE_KEY` (anon).

## 2. Server Deployment
1. Deploy the `server/` directory.
2. Set Environment Variables:
   - `GROQ_API_KEYS`
   - `SUPABASE_URL`
   - `SUPABASE_KEY`
   - Upload `credentials.json` (Google Sheets) to the server or set it as a base64 env var (requires code tweak).

## 3. Client Deployment
1. Deploy `client/`.
2. Update `script.js` line 1: `const API_URL = "https://your-backend-url.com";`
